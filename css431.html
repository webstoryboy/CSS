column-rule-color

column-rule-color 속성은 컬럼의 라인 색을 정의합니다.

특징 설명
기본 값 column-rule-color : correntColor;
상속 안됨
애니메이션 가능
적용 column
버전 CSS3

정의(Definition)

  • column-rule-color 속성은 컬럼의 라인 색을 정의합니다.

문법(Syntax)

column-rule-color : color

/* 키워드 컬러 속성을 사용하는 경우 */
column-rule-color: red;

/* rgba 속성을 사용하는 경우 */
column-rule-color: rgb(46, 23, 0);
column-rule-color: rgba(46, 23, 0, 0.3);

/* hsla 속성을 사용하는 경우 */
column-rule-color: hsl(399, 80%, 20%);
column-rule-color: hsla(399, 80%, 20%, 0.3);

/* 투명도 속성을 사용하는 경우 */
column-rule-color: transparent;

/* 현재 컬러 속성을 사용하는 경우 */
column-rule-color: currentcolor;

/* 글로벌 속성을 사용하는 경우 */
column-rule-color: inherit;
column-rule-color: initial;
column-rule-color: unset;

속성(Property)

속성값 설명
color 컬럼의 색을 설정합니다.

Sample1

콘텐츠 요소의 상하 관계 정렬 상태를 설정하는 예제입니다.

결과

컬럼은 CSS에서 레이아웃을 작성하기 위한 새로운 속성입니다. column-count 속성은 컬럼의 수를 정의하고, column-fill 속성은 열의 지정방법을 정의하고, column-gap 속성은 열의 간격을 정의하고, column-rule 속성은 컬럼 라인의 가로값, 스타일, 컬러를 정의하고, column-rule-color 속성은 컬럼의 라인 색을 정의하고, column-rule-style 속성은 컬럼 라인의 스타일을 정의하고, column-rule-width 속성은 컬럼 라인의 가로값을 정의하고, column-span 속성은 열의 속성을 정의하고, column-width 속성은 컬럼의 가로 값을 정의하고, columns 속성은 열의 폭과 열의 수를 정의합니다.
