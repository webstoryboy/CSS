columns

columns 속성은 열의 폭과 열의 수를 정의합니다.

특징 설명
기본 값 column-width : auto;
column-count : auto;
상속 안됨
애니메이션 안됨
적용 column
버전 CSS3

정의(Definition)

  • columns 속성은 열의 폭과 열의 수를 정의합니다.
  • column-width와 column-count를 같이 사용할 경우 count가 우선 적용됩니다.

문법(Syntax)

columns : auto | column-width | column-count

/* 단위 속성이 있는 경우 : column-width 설정 */
columns: 12em;

/* 단위 속성이 없는 경우 : column-count 설정 */
columns: 3;

/* 두 개의 속성이 있는 경우 : column-count | auto */
columns: 2 auto;

/* 두 개의 속성이 있는 경우 : column-width | auto */
columns: 10em auto;

/* 두 개의 속성이 있는 경우 : auto | auto */
columns: auto auto;

속성(Property)

속성값 설명
auto 컬럼을 자동으로 설정합니다.
column-width 컬럼의 가로 값을 설정합니다.
column-count 컬럼의 갯수를 설정합니다.

Sample1

컬럼의 가로 값을 설정하는 예제입니다.

결과

컬럼은 CSS에서 레이아웃을 작성하기 위한 새로운 속성입니다. column-count 속성은 컬럼의 수를 정의하고, column-fill 속성은 열의 지정방법을 정의하고, column-gap 속성은 열의 간격을 정의하고, column-rule 속성은 컬럼 라인의 가로값, 스타일, 컬러를 정의하고, column-rule-color 속성은 컬럼의 라인 색을 정의합니다. column-rule-style 속성은 컬럼 라인의 스타일을 정의하고, column-rule-width 속성은 컬럼 라인의 가로값을 정의하고, column-span 속성은 열의 속성을 정의하고, column-width 속성은 컬럼의 가로 값을 정의하고, columns 속성은 열의 폭과 열의 수를 정의합니다.
