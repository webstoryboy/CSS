column-gap

column-gap 속성은 컬럼의 간격을 정의합니다.

특징 설명
기본 값 column-gap : normal
상속 안됨
애니메이션 가능
적용 column
버전 CSS3

정의(Definition)

  • column-gap 속성은 열의 간격을 정의합니다.
  • 처음에는 grid-column-gap 속성으로 사용하였지만 현재는 grid가 생략되어 column-gap으로 사용합니다.
  • grid 속성에서 간격을 주기 위해서는 grid-column-gap으로 사용합니다.

문법(Syntax)

column-gap : length | normal

/* 키워드 속성을 사용한 경우 */
column-gap: normal;

/* 단위 속성을 사용한 경우 */
column-gap: 10px;
column-gap: 2.5em;
column-gap: 5%;

/* 글로벌 속성을 사용한 경우 */
column-gap: inherit;
column-gap: initial;
column-gap: unset;

속성(Property)

속성값 설명
length 컬럼의 간격을 CSS 단위로 설정합니다.
normal 컬럼의 간격을 기본 값으로 설정합니다.

컬럼의 간격을 설정하는 예제입니다.

결과

컬럼은 CSS에서 레이아웃을 작성하기 위한 새로운 속성입니다. column-count 속성은 컬럼의 수를 정의하고, column-fill 속성은 열의 지정방법을 정의하고, column-gap 속성은 열의 간격을 정의하고, column-rule 속성은 컬럼 라인의 가로값, 스타일, 컬러를 정의하고, column-rule-color 속성은 컬럼의 라인 색을 정의하고, column-rule-style 속성은 컬럼 라인의 스타일을 정의하고, column-rule-width 속성은 컬럼 라인의 가로값을 정의하고, column-span 속성은 열의 속성을 정의하고, column-width 속성은 컬럼의 가로 값을 정의하고, columns 속성은 열의 폭과 열의 수를 정의합니다.
