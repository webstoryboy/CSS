animation-iteration-count 속성은 애니메이션 반복 횟수 설정합니다.

특징 설명
기본 값 animation-iteration-count : none;
상속 안됨
애니메이션 안됨
적용 animations
버전 CSS3

정의(Definition)

  • animation-iteration-count 속성은 애니메이션 반복 횟수 설정합니다.

문법(Syntax)

animation-iteration-count : number | infinite | inherit;

속성(Property)

속성값 설명
number 애니메이션 반복 횟수를 숫자를 통해 설정합니다.
infinite 애니메이션 반복 횟수를 무한으로 설정합니다.

클릭하면 애니메이션이 동작 횟수를 설정하는 예제입니다.

count1

리셋 클릭
CSS
.count1 {animation: move1 1s 1 ease;}
.count2 {animation: move1 1s 2 ease;}
.count3 {animation: move1 1s 3 ease;}
.infinite {animation: move1 1s infinite ease;} 
@keyframes move1 {
    0%   {left: 0%;}
    50%  {left: calc(100% - 70px);}
    100% {left: 0%;}
}
