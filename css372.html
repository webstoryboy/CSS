animation-play-state

animation-play-state 속성은 애니메이션 진행상태를 설정합니다.

특징 설명
기본 값 animation-play-state : running;
상속 안됨
애니메이션 안됨
적용 animations
버전 CSS3

정의(Definition)

  • animation-play-state 속성은 애니메이션 진행상태를 설정합니다.

문법(Syntax)

animation-play-state : paused | running | inherit;

속성(Property)

속성값 설명
paused 애니메이션의 움직임을 정지시킵니다.
running 애니메이션을 움직이게 합니다.

Sample1

클릭하면 애니메이션이 플레이하고 정지하는 예제입니다.

결과
state
running paused
CSS
.state > div.running {animation: move2 3s 10 ease running;}
.state > div.paused {animation: move2 3s 10 ease paused;}
